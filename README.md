# 🏰 LuxeEstate Marketplace

Une plateforme immobilière premium conçue pour offrir une expérience de recherche de propriétés de luxe fluide et moderne.

## 📖 À propos du projet

**LuxeEstate** est une application web sophistiquée qui permet aux utilisateurs de parcourir des propriétés prestigieuses à travers le monde. Le projet met l'accent sur une interface utilisateur élégante (UI) et une expérience utilisateur (UX) intuitive, facilitant la recherche de "sanctuaires de rêve".

### ✨ Fonctionnalités clés
*   **Filtrage Avancé** : Système de recherche précis par ville, quartier ou titre.
*   **Galeries d'Images** : Présentation haute résolution des propriétés architecturales.
*   **Favoris** : Possibilité pour les utilisateurs de sauvegarder leurs propriétés préférées.
*   **Design Responsive** : Optimisé pour une utilisation sur mobile, tablette et ordinateur.

## 🛠️ Stack Technique

*   **Frontend** : [React](https://reactjs.org/) + [Vite](https://vitejs.dev/)
*   **Styling** : [Tailwind CSS](https://tailwindcss.com/)
*   **IA Engine** : Généré et optimisé via Gemini 3 Flash
*   **Déploiement** : Prêt pour Vercel / Google Cloud Run
